# Проект Space

Этот репозиторий создан в рамках практической работы №5 по организации работы в GitHub.  
Цель работы — освоить базовые навыки работы с системой контроля версий Git и веб-сервисом GitHub.

## Содержание репозитория

- **main** — основная ветка с начальными файлами проекта.
- **ВтораяВетвь** — ветка с дополнительными файлами.
- **ТретьяВетвь** — ветка с текстовым файлом о назначении GitHub.

## Используемые технологии
